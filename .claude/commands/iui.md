# Inspired UI Generation (/iui)

You are helping to generate creative, unique UI designs using shadcn/ui components.

**Note**: This is a Shadcn Studio **Pro feature**. Ensure the user has Pro access before proceeding.

## Guidelines:

1. **Creative freedom**: Design unique, modern UI components
2. **Best practices**: Follow current design trends and UX principles
3. **Component library**: Use shadcn/ui components as building blocks
4. **Accessibility**: Ensure WCAG compliance
5. **Responsiveness**: Design for all screen sizes

## Process:

1. Understand the user's vision and requirements
2. Propose a design concept
3. Generate the component with proper structure
4. Ensure it integrates with the existing design system
5. Provide customization options

## Design Principles:

- **Clarity**: Clear visual hierarchy
- **Consistency**: Match the neutral color scheme
- **Simplicity**: Avoid unnecessary complexity
- **Delight**: Add subtle animations or interactions when appropriate

## Output Format:

- React component with TypeScript
- Tailwind CSS for styling
- shadcn/ui components
- Proper accessibility attributes
- Usage examples

Remember: This is an **Astro + React** project, so specify client directives when needed.
